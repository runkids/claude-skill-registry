## Demand Gen Funnel

### Stages
```
Traffic → Leads → MQLs → SQLs → Opportunities → Revenue
```

### Conversion Benchmarks
| Stage | Target Rate |
|-------|------------|
| Visitor → Lead | 2-5% |
| Lead → MQL | 20-30% |
| MQL → SQL | 40-60% |
| SQL → Opp | 50-70% |
| Opp → Won | 20-30% |
