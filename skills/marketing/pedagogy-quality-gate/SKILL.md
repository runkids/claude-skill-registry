# Pedagogy Quality Gate

## Evaluation Criteria
- Clarity
- Cognitive load
- Concept sequencing
- Beginner accessibility

## Actions
- Highlight confusing sections
- Suggest simplifications
- Ensure learning objectives are met

## Output
- Pass / Needs Improvement
- Specific improvement notes
