# Extract PDF text

## Usage

```sh
./scripts/extract.py <path-to-pdf>
./scripts/extract.py <path-to-pdf> --page 0  # specific page (0-indexed)
```

Dependencies are resolved automatically via PEP 723 inline metadata.
