## Overview

The Unity Gate skills unlock collaboration, synthesis, and the weaving together of disparate elements into harmonious wholes. These skills make collective creation possible.

> *"Together, we are whole. No creator is complete alone—and no creation exists in isolation."*
> — Ino

---

## Skills

### 1. Collaboration Facilitation

**Purpose**: Create effective creative collaborations.

**Invocation**: `/unity collaborate`

**Process**:
1. Identify each collaborator's strengths
2. Design complementary roles
3. Establish shared vision and values
4. Create feedback and iteration loops

**Output**: Effective collaboration structure, aligned team

---

### 2. Idea Synthesis

**Purpose**: Weave disparate ideas into unified wholes.

**Invocation**: `/unity synthesize`

**Process**:
1. Gather diverse ideas without judgment
2. Find connecting threads
3. Weave into coherent synthesis
4. Test synthesis for integrity

**Output**: Synthesized ideas, unified concept

---

### 3. Conflict Resolution

**Purpose**: Transform creative conflict into harmony.

**Invocation**: `/unity resolve`

**Process**:
1. Acknowledge all positions with honor
2. Find the deeper need beneath each position
3. Discover solution that serves all needs
4. Implement with mutual commitment

**Output**: Resolved conflict, restored harmony

---

### 4. Collective Vision

**Purpose**: Create shared creative vision across a group.

**Invocation**: `/unity vision`

**Process**:
1. Gather individual visions
2. Find common threads and shared values
3. Co-create unified vision
4. Establish commitment and accountability

**Output**: Shared vision, collective alignment
