# Android Patterns

Android-specific best practices for Kotlin and Jetpack Compose development.

## When This Applies

- Building Android applications
- Using Jetpack Compose or Views
- Kotlin coroutines

## Extends

This plugin extends **`platform-mobile`** (generic mobile patterns). Apply those rules first, then these Android-specific ones.

## Quick Reference

| Section | Impact | Prefix |
|---------|--------|--------|
| Compose | HIGH | `compose-` |
| Coroutines | HIGH | `coroutine-` |
| Architecture | MEDIUM | `arch-` |
