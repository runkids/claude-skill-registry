# Image Generation Skill

Generate images using AI for mockups, icons, and visual assets.

## Use Cases

- UI mockups and wireframes
- Custom icons and graphics
- Marketing visuals
- Placeholder images
- Concept art

## Example Prompts

```bash
"generate image: modern SaaS dashboard mockup
 Style: Clean, minimal, Tailwind aesthetic
 Colors: Blue/gray palette
 Elements: Sidebar, charts, data tables
 Save: temp/images/dashboard-mockup.png"
```

```bash
"create 10 custom icons for navigation:
 - Home, Search, Settings, Profile, Notifications, etc.
 Style: Outline, 24x24px, stroke-2
 Format: SVG
 Save: src/assets/icons/"
```

## Integration

Generated images can be:
- Used as design inspiration
- Converted to code (via screenshot → Gemini)
- Used as placeholders
- Refined by designer

---

**Note**: AI-generated images work best as starting points for human refinement.
