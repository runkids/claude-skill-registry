# Geolocation

## Overview

Location patterns for verifying tasker presence at task locations using Expo Location.

## When to Use

- Tracking tasker location during task execution
- Verifying tasker is within task radius (geofencing)
- Validating GPS accuracy before accepting location
- Implementing anti-spoofing measures
