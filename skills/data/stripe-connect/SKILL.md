# Stripe Connect

## Overview

Stripe Connect patterns for marketplace payments where COVER facilitates transactions between clients and taskers.

## When to Use

- Creating Connected Accounts for taskers
- Processing payments with escrow (auth → capture)
- Handling application fees (COVER commission)
- Managing refunds and disputes
- Setting up Stripe webhooks
