## Purpose
Confirm whether to continue into refactoring after mapping.

## Steps
1. Ask if the user wants to start refactoring now.
2. If yes, create/update `PLANS.md` and `IMPACT.md`, then proceed.
3. If no, capture refactor ideas in `BACKLOG.md` and stop.
4. Ask only once and keep it short.
