# Common Tools

全ロールで利用可能な基本ツールセット。

## 利用可能なツール

- **get_skill**: スキルの詳細情報（SKILL.md）を取得
- **get_resource**: スキルに含まれるリソースファイルを取得
- **run_script**: スキルに含まれるスクリプトを実行
