# Frontend Design Patterns

Web UI design best practices with Tailwind CSS. Extends generic `design` principles with web-specific implementation.

## When This Applies

- Building web UIs with Tailwind CSS
- Implementing responsive designs
- Working with design tokens

## Extends

This plugin extends **`design`** (universal design principles). Apply those rules first, then these web-specific ones.

## Quick Reference

| Section | Impact | Prefix |
|---------|--------|--------|
| Design Tokens | HIGH | `token-` |
| Responsive | HIGH | `responsive-` |
| Layout | MEDIUM | `layout-` |
| Typography | LOW | `typography-` |

## Rules

See `rules/` directory for individual rules organized by section prefix.
