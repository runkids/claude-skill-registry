# Git-Bug Issue Tracker

Use `git bug` to manage issues directly in the repository.

## Common Commands

- **List issues**: `git bug ls`
- **Create new issue**: `git bug new`
- **Show issue details**: `git bug show <id>`
- **Add comment**: `git bug comment <id>`
- **Edit issue**: `git bug edit <id>`
- **Change status**: `git bug status <id> <new-status>`
- **Sync issues**: `git bug pull` and `git bug push`
