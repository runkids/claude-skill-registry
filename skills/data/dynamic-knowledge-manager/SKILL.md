# Dynamic Knowledge Manager

Use this skill to update indices and manage knowledge drift.

## Workflow

1) Identify stale or conflicting entries.
2) Update or deprecate records with provenance.
3) Rebuild indexes if required.
4) Report deltas and stability impact.

## Output expectations

- Provide update summary and index status.
