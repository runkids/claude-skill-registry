## Brand Guidelines

### Colors

**Primary Colors:**

- Forest Green: `#2C5F2D` - Growth, stability, trust, natural materials. Use for primary brand color, headers, calls-to-action
- Warm Clay: `#D4A574` - Earthiness, craftsmanship, warmth, local materials. Use for accents, highlights, warmth in designs

**Secondary Colors:**

- Steel Gray: `#6B7280` - Professionalism, strength, modern materials. Use for supporting text, borders, neutral backgrounds
- Clean White: `#FFFFFF` - Clarity, transparency, simplicity. Use for backgrounds, breathing room, digital interfaces
- Deep Charcoal: `#1F2937` - Sophistication, strength, premium quality. Use for primary text, strong statements, premium products

**Accent/Utility Colors:**

- Alert Red: `#DC2626` - Warnings, urgent actions
- Success Green: `#10B981` - Confirmations, positive actions
- Info Blue: `#3B82F6` - Information, links, interactive elements

### Typography

**Primary Font (Body Text):**
- **Noto Sans Thai + Noto Sans** - Clean, modern, excellent Thai language support
- Use for body text, general content, website, apps
- Professional, highly readable, works across digital and print

**Heading Font:**
- **Prompt (Thai) + Inter** - Bold, modern, designed for Thai language
- Use for headers, titles, emphasis
- Strong visual hierarchy, friendly yet professional

### Language

- **Primary Language**: Thai (standard Thai for website, respect for regional dialects)
- **First Store Location**: กุมภวาปี, อุดรธานี (Kumphawapi, Udon Thani)
- **Domain**: sanwatsadu.com

## Features

### Smart Font Application

- Applies Prompt font to headings
- Applies Noto Sans Thai to body text
- Automatically handles Thai language rendering
- Preserves readability across all systems

### Text Styling

- Headings: Prompt Bold
- Body text: Noto Sans Thai
- Smart color selection based on background
- Preserves text hierarchy and formatting

### Color Application

- Primary actions and headers use Forest Green (`#2C5F2D`)
- Accents and highlights use Warm Clay (`#D4A574`)
- Text uses Deep Charcoal (`#1F2937`) on light backgrounds
- Maintains visual warmth while staying professional

## Design Principles

- **Clarity Over Complexity:** Every design choice should serve the user's understanding
- **Breathing Room:** White space is not wasted space—it creates calm and focus
- **Consistent Visual Hierarchy:** Users should immediately know what's important
- **Warm Technology:** Digital doesn't mean cold. Bring humanity into every interface
- **Accessible for All:** High contrast, readable fonts, clear labels for everyone

## Technical Details

### Font Management

- Uses system-installed Noto Sans Thai and Prompt fonts when available
- Provides automatic fallback to system sans-serif fonts
- Excellent Thai language support
- For best results, pre-install Noto Sans Thai, Noto Sans, Prompt, and Inter fonts

### Color Application

- Uses RGB/HEX color values for precise brand matching
- Forest Green RGB: 44, 95, 45
- Warm Clay RGB: 212, 165, 116
- Deep Charcoal RGB: 31, 41, 55
- Maintains color fidelity across different systems
