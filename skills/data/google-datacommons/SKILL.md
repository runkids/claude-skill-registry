# Google Data Commons Skill

## Overview

Public dataset access via Data Commons. 90%+ context savings.

## Tools (Progressive Disclosure)

### Query

| Tool       | Description          |
| ---------- | -------------------- |
| sparql     | Execute SPARQL query |
| search     | Search entities      |
| get-entity | Get entity details   |

### Statistics

| Tool        | Description           |
| ----------- | --------------------- |
| get-stat    | Get statistical value |
| time-series | Get time series data  |
| place-stats | Get stats for place   |

### Variables

| Tool           | Description           |
| -------------- | --------------------- |
| list-variables | List stat variables   |
| variable-info  | Get variable metadata |

## Common Datasets

- Demographics (population, age, gender)
- Economics (GDP, unemployment, income)
- Health (disease rates, mortality)
- Environment (emissions, climate)
