# Textbook Structure Generator

## Responsibilities
- Convert syllabus into:
  - Modules
  - Weeks
  - Chapters
- Generate sidebar-compatible structure

## Rules
- One concept per chapter
- Progressive difficulty
- Clear dependency order

## Output
- Folder paths
- Chapter titles
- Learning goals per chapter
