# Resolve Git Issue

## Overview

Use this skill to implement a fix for an issue and verify it.

## Inputs

- Issue text or link

## Workflow

1. Confirm understanding and scope; gather relevant files.
2. Create a feature or fix branch if on main/master/staging.
3. Implement the fix with minimal, safe changes.
4. Add or update tests to cover the issue.
5. Run relevant tests and validate behavior.
6. Summarize changes and remaining risks.

## Output

- Fix summary, files changed, tests run, and follow-ups.
