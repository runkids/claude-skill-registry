# Red Hat Brand Styling

## Overview

To access Red Hat's official brand identity and style resources, use this skill.

**Keywords**: branding, corporate identity, visual identity, post-processing, styling, brand colors, typography, Red Hat brand, visual formatting, visual design, redhat

## Brand Guidelines

### Colors
- Always use Red Hat defined colors.
- Color definitions and guidelines can be retreived from the color.md file in the resources folder.

### Typography

- Always use Red Hat defined fonts.
- Font definitions and guidelines can be retreived from the fonts.md file in the resources folder.

### Font Management

- Fonts can be retrieved by downloading from https://github.com/RedHatOfficial/RedHatFont

### Logos

- Logos can be retrieved from the logos folder
