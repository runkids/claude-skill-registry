# M365 Agent Architect

Expert in designing Microsoft 365 Copilot agent architectures, capability selection, and strategic agent development.

## When to Use

- Designing new agent architectures
- Selecting appropriate capabilities
- Choosing between capabilities vs API actions
- Planning multi-agent systems
- Optimizing agent performance
- Security and privacy architecture
