# Event Management Skill

## Overview
Plan, execute, and measure developer events that create meaningful connections and drive community growth.

## Core Topics

### Event Types
- Conferences and summits
- Meetups and user groups
- Hackathons and code jams
- Workshops and training sessions
- Virtual and hybrid events

### Event Planning
- Budgeting and sponsorship
- Venue and logistics
- Speaker selection and management
- Marketing and promotion

### Event Execution
- Day-of coordination
- Live streaming and recording
- Attendee engagement
- Crisis management

### Post-Event Activities
- Content repurposing (recordings, slides)
- Feedback collection and analysis
- Follow-up campaigns
- ROI measurement

## Learning Objectives
- Plan events of various scales
- Execute flawless event logistics
- Maximize attendee engagement
- Measure and improve event ROI
