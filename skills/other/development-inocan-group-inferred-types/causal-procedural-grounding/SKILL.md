# Causal Procedural Grounding

Use this skill to connect causes to validated remediations.

## Workflow

1) Identify causal chain and failure modes.
2) Map causes to verified actions.
3) Execute or recommend with safeguards.
4) Store causal evidence trail.

## Output expectations

- Provide causal map and action linkage.
