# Scout Refine

## Overview

Run Scout to refresh context, then Refine the plan.

## Inputs

- Plan file path
- Optional feature description

## Workflow

1. Run Scout to identify relevant files and docs.
2. Run Refine with the existing plan and new context.

## Output

- Updated plan file and change summary
