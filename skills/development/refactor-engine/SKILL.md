# Refactor Engine

Use this skill to run safe refactors with validation.

## Workflow

1) Identify refactor target and impact graph.
2) Break into minimal change sets.
3) Apply changes with tests or static checks.
4) Summarize diffs and risks.

## Output expectations

- Provide a refactor plan and verification steps.
